# Scope Kanal Eğeleri

Kanal eğeleri ürün kataloğu ve Telegram sipariş bildirim sistemi.

## 🛠 Teknolojiler
- **Backend:** Go (Gin Framework)
- **Veritabanı:** PostgreSQL (GORM)
- **Frontend:** HTML, CSS, JS
- **Bildirim:** Telegram Bot API
- **Konteyner:** Docker

## 🚀 Kurulum
1. `.env` dosyasını hazırlayın.
2. Uygulamayı başlatın:
   ```bash
   docker-compose up -d --build
   ```

## ✨ Özellikler
- Ürün Listeleme & Detay
- Admin Paneli (Ürün Yönetimi)
- Telegram Sipariş Bildirimi
