### ✨ Temel Özellikler

- 🎯 **Dinamik Katalog:** GORM ve PostgreSQL ile güçlendirilmiş hızlı ürün yönetimi.
- 📱 **Modern UI:** Bootstrap 5 ile her cihazda kusursuz görünüm.
- ✉️ **Telegram Entegrasyonu:** Siparişlerin anlık olarak Telegram kanalına düşmesi.
- 🐳 **Dockerize Yapı:** Tek komutla (`docker-compose up`) anında kurulum.
